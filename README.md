# Python Data Analysis Dashboard

## Project Overview

This project demonstrates basic data analysis using Python, Pandas, and Matplotlib.

The project loads a Titanic dataset, performs data exploration, cleans missing values, generates summary statistics, filters data, groups information, creates visualizations, and exports a cleaned dataset.

---

## Features

- Load CSV dataset
- Display first five rows
- Dataset information
- Summary statistics
- Count passengers
- Count survivors
- Average, minimum and maximum age
- Gender distribution
- Passenger class distribution
- Filtering
- Sorting
- GroupBy analysis
- Missing value detection
- Missing value handling
- Export cleaned dataset
- Bar Chart
- Pie Chart
- Histogram

---

## Technologies Used

- Python 3
- Pandas
- Matplotlib

## How to Run

Clone the repository

```bash
git clone <repository-url>
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the project

```bash
python main.py
```
